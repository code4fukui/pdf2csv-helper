# pdf2csv-helper
日本語のREADMEはこちらです: [README.ja.md](README.ja.md)

テキストからCSVテーブルをコピーできるシンプルなWebアプリです。

## 機能
- テキストからCSVテーブルを抽出する
- テーブルをクリップボードにコピーする
- PDFテキストからテーブルを抽出するのに使用できる

## 使用方法
1. CSV表を含むテキストを入力またはペーストしてください。
2. "CSV tableをテキストからコピー"ボタンをクリックしてください。
3. CSVテーブルがクリップボードにコピーされ、スプレッドシートやその他のアプリケーションに貼り付けることができます。
